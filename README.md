# apollo-datastore-rest
ReST version of my apollo project.

uses the following:
- Google App Engine
- Jersey (JAX-RS)
- JSP MVC Template (Jersey module)
- Jackson (JSON library)

Data Access methods:
- Objectify (App Engine Datastore)
- JDO (App Engine Datastore ORM)

To be implemented:
- Apache Oltu Oauth2 protection
